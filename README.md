lampcake
===

## Composition.
- PHP 7.1
    - Composer 1.*
    - CakePHP 3.*
- Apache 2.2
- MySql 5.6
- Linux CentOS 6.7 (Production: 6.5)
    - yum
        - epel
        - rpmforge
        - remi
- Node.js 8.*
    - npm 5.*


## Commands.
```
# build
$ vagrant up

# rebuild
$ vagrant reload

# ssh to vm
$ vagrant ssh
```
